University Review Website

The University Review Website compliments the UReview app with the statistical inforamation of over 2000 universities in the US. The academics are displayed in a simple, user friendly way so navigating through college statistics is not an overwelhming process. This website includes a interavtice U.S. map for an easy search for all universities.

Helping Hands Application

The Helping Hands App Displays two interactive maps, using the Google Maps API. This application was created to allow users to pinpoint a homeless person and add an item they could benefit from; Example: blanket. The database stores and displays the latitude/longitude/item of each marked homeless person for each user to view. The Homepage is a scrolling parallax.
